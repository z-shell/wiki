如果在源代码仓库中配置了 `zunit` 和 `make` 测试，附件就会运行这些测试。

<ImgShow
  width={1000}
  height={900}
  img='https://user-images.githubusercontent.com/59910950/162143845-c44ead50-b21a-46c0-8372-18325eb1f33a.gif'
  alt='Annex - z-a-test preview'
/>

只需要像任何其他插件一样加载它以使其激活：

```shell
zi light z-shell/z-a-test
```

<details>
  <summary>📖 <b>配置</b></summary>

要以详细方式运行测试，请运行：

```shell
zstyle :zi:annex:test quiet 0
```

要在安装或更新前跳过单个插件的测试，请添加 `notest` ice 修饰符。

```shell showLineNumbers
zi ice notest
zi load …
```

</details>
